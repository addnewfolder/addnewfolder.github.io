行為改變科學的實務設計：人類採取特定行動的原因

前言

每個人都是獨特的個體，處於不同的環境中。人類的決策過程複雜、混亂且充滿驚奇。為了設計能夠改變行為的產品，我們需要擁抱並接受人類心智的運作方式，以及心智與環境的互動。這樣才能更深入地了解如何有效地改變行為。

產品如何鼓勵使用者行動

當以下條件同時發生時，使用者更可能喜歡並採用該產品：

  1. 線索（Cue）：觸發使用產品的可能性。
  2. 反應（Reaction）：使用產品時的瞬間感受和使用後的感覺。
  3. 評估（Evaluation）：對行動的效益和成本進行思考。
  4. 能力（Ability）：產品的可用性和易用性。
  5. 時機（Timing）：使用者是否會立即行動還是選擇稍後。

1. 線索（Cue）

  • 外部線索的重要性：在產品設計中，外部線索（外界環境的觸發）比內部線索（使用者的自省或幻想）更為重要。
  • 融入日常生活：將產品融入使用者的日常生活中，創造多樣的情境，與使用者的規律生活建立強連結。
  • 避免干擾：確保線索出現的時間不會讓使用者過於分心，影響對產品的注意力。

2. 反應（Reaction）

  • 瞬間評估：使用者在使用產品的瞬間會進行評估，需要關注以下方面：
  • 信任感：建立使用者對產品的信任。
  • 行為觀察：注重使用者的實際行為，而非他們的自我描述。
  • 良好初體驗：確保使用者的第一次使用體驗是積極的。

3. 評估（Evaluation）

  • 教育使用者：傳達行動的好處，增強他們的動機。
  • 即時獎勵：使用金錢或其他獎勵來激勵，但要立即提供，避免使用者等待替代方案。
  • 以使用者為中心：價值評估應基於使用者的觀點，而非設計者的假設。

4. 能力（Ability）

為了使使用者能夠立即行動，需要消除以下障礙，或針對特定使用者群體：

  • 行動計畫：提供清晰的行動步驟，讓使用者了解如何使用產品。
  • 資源需求：確保使用者具備必要的資源，如金錢、體力、設備或技術能力。
  • 技能要求：避免需要高深技能或複雜的使用介面。
  • 信心建立：增強使用者對成功完成行動的信心。

5. 時機（Timing）

  • 立即行動的重要性：延遲行動可能導致使用者放棄，因此產品應促使他們立即採取行動。
  • 處理無聊或低緊急性情境：在這些情況下，提供有吸引力的行動選項，以降低他們選擇替代方案的可能性。
  • 特定時間點：設定具體的時間或情境，讓使用者感到現在是行動的最佳時機。
  • 強化承諾感：幫助使用者實現「說到做到」，避免他們忘記或推遲行動。

CREATE 行動漏斗

由於以下原因，許多有意行動的使用者可能被過濾掉或分心去做其他事情，因此產品需要在每個階段都勝過其他競爭行為：

  1. 缺乏線索：未注意到觸發行動的線索。
  2. 負面反應：對產品的即時反應不佳。
  3. 效益不足：認為行動的成本高於收益。
  4. 能力不足：缺乏立即行動的能力或資源。
  5. 緊急性不足：沒有足夠的緊迫感促使行動。

BJ Fogg 的觀點

  • 階段間的互補：即使某個階段較弱，其他階段的強勢也可能彌補這一劣勢。
  • 動態過程：使用者每次通過 CREATE 漏斗的機率都不同，因為他們的心境和環境會發生變化。

結論

透過理解人類採取特定行動的原因，設計者可以創造出更有效的產品，促進行為改變。在設計過程中，需要全面考慮使用者的心理和環境因素，並在關鍵階段提供支持，才能成功引導他們採取期望的行動。
